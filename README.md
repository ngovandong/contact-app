## Installation

_Below is an example of how you can instruct your audience on installing and setting up your app.

1. Clone the repo
   ```sh
   https://github.com/ngovandong/BigTwoGameMobile
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Install Expo 
   ```js
   npm install --global expo-cli
   ```
4. Start
   ```js
   expo start
   ```
